# tinywordle
 k - key event
 p - print to pre function (what to append, set word entry)
 l - letter entry
 q - pre element
 w - word entry
 h - check word function
 d - daily word
 x - list of word entry characters
 a - number of attempts
 g - length of word entry
 s - backspace function
 '#' = not in the word, '!' = yes, '?' = in the word
